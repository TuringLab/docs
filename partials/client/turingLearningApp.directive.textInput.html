textInput
directive in module turingLearningApp

Description

Dropdown input to dispaly a list of options which can be selected. When a value is selected the parameter is chanaged and the callback is called.

Usage

as element:
<text-input
       text="{string}"
       [format="{string}"]>
</text-input>

Parameters

ParamTypeDetails
textstring

The text to be edited

format
(optional)
string

Text format for syntax highlighting

(default: 'plain')

Environment
service in module turingLearningApp

Description

Service used to control the development environment which is also known as the editor. This service is used to receive changes in size experienced by the developmentPlaceholder and communicate the required size to the developmentEnvironment

Methods

  • set_environment_listener(_listener)

    Defines the callback function to be called to notify the developmentEnvironment that the environment has changed

    Parameters
    ParamTypeDetails
    _listenerfunction

    Function to be called on environment change

  • update_size(width, height)

    Updates the width and height of the environment and calls the listener function to notify the developmentEnvironment that the size of the development environment has changed

    Parameters
    ParamTypeDetails
    widthnumber

    New width of the development environment

    heightnumber

    New height of the development environment

  • update_source(source)

    Updates the source of the environment and calls the listener function to notify the developmentEnvironment that the source of the editor has changed

    Parameters
    ParamTypeDetails
    sourcestring

    New url source of development environment

History
service in module turingLearningApp

Description

Service used to record the states which have been visited using the learning system to allow users to navigate back up the tree and know the parent of a given brick

Dependencies

  • $state
  • $localStorage

Methods

  • phantom(params)

    Make a phantom state change which does not notify or reload the state but changes the state parameters. This replaces the previous state in the history

    Parameters
    ParamTypeDetails

    object

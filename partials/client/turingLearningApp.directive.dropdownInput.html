dropdownInput
directive in module turingLearningApp

Description

Dropdown input to dispaly a list of options which can be selected. When a value is selected the parameter is chanaged and the callback is called.

Usage

as element:
<dropdown-input
       selected
       options="{string}"
       [label="{string}"]
       [callback="{function}"]>
</dropdown-input>

Parameters

ParamTypeDetails
selectedstring

The currently selected value of options

optionsstring

Options which can be selected using dropdown

label
(optional)
string

Label for dropdown input to be shown to the left

callback
(optional)
function

Function to be called when selected is changed

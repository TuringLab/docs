roundButton
directive in module turingLearningApp

Description

Simple round button with given icon and text which will call click function when clicked

Usage

as element:
<round-button
       click="{function}"
       icon="{string}"
       text="{string}">
</round-button>

Parameters

ParamTypeDetails
clickfunction

Function to be called when button clicked

iconstring

Icon class to dispaly on button

textstring

Text label to show on button

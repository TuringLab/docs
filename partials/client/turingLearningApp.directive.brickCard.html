brickCard
directive in module turingLearningApp

Description

Directive used to display a brick. The brick will be displayed with a header which contains the purpose or signal of the brick along with any action which is attached to the brick. The brick will also be displayed with a footer containing interactions

Dependencies

  • $scope
  • Brick

Usage

as element:
<brick-card
       brick="{Object}"
       open-brick="{Boolean}"
       [number="{Number}"]>
</brick-card>

Parameters

ParamTypeDetails
brickObject

The brick object to display

openBrickBoolean

Function called when the brick is clicked

number
(optional)
Number

Number to be displayed in brick header
